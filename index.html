About Me

Nama saya Putri Aulia Maulidina, saya lahir di Bandung, 16 Mei 2003, hobi saya menonton film. Saya bersekolah di SDN Gadis 1, SMPN 1 Ciparay, SMAN 1 Ciparay. Sekarang saya kuliah di Universitas Pasundan jurusan Teknik Informatika.

Pengalaman organisasi saya adalah, saat di bangku SMA saya mengikuti organisasi MPK, saya pernah mengikuti panitia pemilihan ketua OSIS, panitia hari karir kelas 12, dan panitia masa pengenalan lingkungan sekolah.
